# GEO & AI Search Literature Review — Organization Memo
## Purpose
This memo defines the folder structure and PDF naming convention for the GEO / AI Search literature review. The goal is to keep the review folder professional, searchable, and easy to share with collaborators.
## Naming Convention
Use the following format for all paper PDFs:
```text
Number_Year_FirstAuthor_ShortTitle.pdf
```
Example:
```text
1_2024_Aggarwal_Generative_Engine_Optimization.pdf
```
**Rationale:** This format preserves the reading order while adding academic metadata such as year and first author.
## Recommended Folder Structure
```text
GEO_AI_Search_Literature_Review/
```

## Practical Rule
Each paper should have one primary folder only. If a paper overlaps multiple themes, keep the primary folder based on the main research contribution and record secondary themes in the literature table or notes.
## Recommended Use
- Store this memo in `00_Memo/`.
- Store the updated CSV in `01_Literature_Table/`.
- Keep the file names exactly aligned with the `PDF File Name` column in the literature table.
